# 🚀 tiny-app-cli
自动提审与发布微信、支付宝小程序

## 安装
With NPM:
```bash
npm install -g tiny-app-cli
```

With PNPM:
```bash
pnpm install -g tiny-app-cli
```

With Yarn:
```bash
yarn add -g tiny-app-cli
```

## 如何使用？
```bash
tiny-app-cli
```
然后按照提示去做!

## 注意事项
- 由于当前功能不完善，在微信小程序提审的过程中，请确保你已经提审过一次小程序
